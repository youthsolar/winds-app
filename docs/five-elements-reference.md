# 五行炁 對應參考表

五行（木、火、土、金、水）所對應的顏色、水晶與精油彙整。供占卜推薦、商品分類、能量調理用途參考。

## 對應總表

| 五行 | 顏色 | 水晶 | 精油 |
|------|------|------|------|
| **木** | 青色、綠色 | 綠幽靈、東陵玉、綠髮晶、孔雀石、橄欖石、葡萄石 | 茶樹、尤加利、絲柏、松針、薄荷、迷迭香 |
| **火** | 紅色、紫色 | 紅紋石、紅碧璽、石榴石、紅瑪瑙、太陽石、紫水晶 | 肉桂、丁香、薑、玫瑰、天竺葵、依蘭 |
| **土** | 黃色、咖啡色 | 黃水晶、虎眼石、黃玉、琥珀、髮晶、土耳其石 | 廣藿香、岩蘭草、檀香、安息香、乳香、雪松 |
| **金** | 白色、金色、銀色 | 白水晶、白幽靈、月光石、銀曜石、拓帕石、賽黃晶 | 薰衣草、白千層、佛手柑、檸檬、葡萄柚、尤加利 |
| **水** | 黑色、藍色 | 黑曜石、藍寶石、海藍寶、青金石、堇青石、拉長石 | 茉莉、橙花、沒藥、洋甘菊、香蜂草、廣藿香 |
